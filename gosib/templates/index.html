go.stillinbeta.com*

a human-readble, single-use URL shortener for moving long URLs between devices

*pretend the domain name is better

Download the Android App (beta)

http://go.stillinbeta.com

1. Shorten

Use the website or the Android app to shorten an unwieldy URL.

2. Read

go.stillinbeta.com will produce a concise, human-readable URL for you to use.

3. Type

Type the URL into another device to transfer it over. No software required.

4. Forget

URLs can only be used once, otherwise we'd run out of words!
